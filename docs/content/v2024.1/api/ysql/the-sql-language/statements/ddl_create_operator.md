## Synopsis

Use the `CREATE OPERATOR` statement to create an operator.

## Syntax

{{%ebnf%}}
  create_operator,
  operator_option
{{%/ebnf%}}

## Semantics

See the semantics of each option in the [PostgreSQL docs][postgresql-docs-create-operator].

## Examples

Basic example.

```plpgsql
yugabyte=# CREATE OPERATOR @#@ (
             rightarg = int8,
             procedure = numeric_fac
           );
yugabyte=# SELECT @#@ 5;
```

```
 ?column?
----------
      120
```
