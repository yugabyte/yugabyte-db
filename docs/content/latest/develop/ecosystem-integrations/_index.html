--- title: Ecosystem Integrations linkTitle: Ecosystem Integrations description: Ecosystem Integrations headcontent: YugaByte DB is driver compatible with Apache Cassandra, Redis and PostgreSQL (beta) clients. It can therefore work with the ecosytem of projects built around these APIs. This page lists some of the ecosystems that YugaByte DB can work with. image: /images/section_icons/develop/ecosystem-integrations.png aliases: - /develop/ecosystem-integrations/ menu: latest: identifier: ecosystem-integrations parent: develop weight: 570 ---
Apache Spark
Powerful open source processing engine built for fast analytics on top of YugaByte DB's Cassandra-compatible YCQL API.
JanusGraph
Scalable graph API that uses Cassandra-compatible YugaByte DB as it's underlying distributed database.
KairosDB
Fast time series metrics API that uses Cassandra-compatible YugaByte DB as it's underlying distributed database.
Presto
Ad-hoc interactive SQL queries on top of Cassandra-compatible YugaByte DB.
Metabase
Ad-hoc interactive SQL queries on top of PostgreSQL-compatible YugaByte DB.
