## v2.20.0.0 - November 13, 2023 {#2.20.0.0}

**Build:** `2.20.0.0-b76`

**Third-party licenses:** [YugabyteDB](https://downloads.yugabyte.com/releases/2.20.0.0/yugabytedb-2.20.0.0-b76-third-party-licenses.html), [YugabyteDB Anywhere](https://downloads.yugabyte.com/releases/2.20.0.0/yugabytedb-anywhere-2.20.0.0-b76-third-party-licenses.html)

### Docker

```sh
docker pull yugabytedb/yugabyte:2.20.0.0-b76
```

### Highlights

- Improved safety checks for rolling upgrades.
  Before a node is upgraded, YBA now checks for under-replicated tablets to make sure your universe is in a healthy state before proceeding with the upgrade.

- Restore history
  After performing a restore, you can now verify the restore details from the restore history tab.

- OIDC
