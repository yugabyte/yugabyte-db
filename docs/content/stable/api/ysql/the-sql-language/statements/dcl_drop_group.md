## Synopsis

Use the `DROP GROUP` statement to drop a role. `DROP GROUP` is an alias for [`DROP ROLE`](../dcl_drop_role) and is used to drop a role.

## Syntax

{{%ebnf%}}
  drop_group
{{%/ebnf%}}

## Semantics

See [`DROP ROLE`](../dcl_drop_role) for more details.

## Example

- Drop a group.

```plpgsql
yugabyte=# DROP GROUP SysAdmin;
```
