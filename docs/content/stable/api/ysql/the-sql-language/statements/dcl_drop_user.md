## Synopsis

Use the `DROP USER` statement to drop a user or role. `DROP USER` is an alias for [`DROP ROLE`](../dcl_drop_role) and is used to drop a role.

## Syntax

{{%ebnf%}}
  drop_user
{{%/ebnf%}}

## Semantics

See [`DROP ROLE`](../dcl_drop_role) for more details.

## Example

- Drop a user.

```plpgsql
yugabyte=# DROP USER John;
```
