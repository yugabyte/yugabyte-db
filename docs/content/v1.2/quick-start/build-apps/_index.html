Build an app description: Build an application on top of YugabyteDB. headcontent: Build an app in the programming language of your choice. image: /images/section_icons/develop/api-icon.png block_indexing: true menu: v1.2: identifier: build-apps parent: quick-start weight: 150 --- An application interacts with YugabyteDB using of a client library, also known as a client driver. Client drivers handle all interaction with the database in a language appropriate to the application. Here is a list of client drivers in various languages for YugabyteDB for building transactional, internet-scale apps. Note that YugabyteDB's YSQL and YCQL APIs are wire compatible with PostgreSQL and Cassandra Query Language respectively. As a result, we will be leveraging the client drivers of these popular languages to connect to YugabyteDB.

We will also be integrating YugabyteDB with the most popular ORM frameworks for each of these languages.
